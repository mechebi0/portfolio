Featured Project

Capstone Project April 2026

Kali → Windows Attack Simulation with Real-Time Sysmon Detection

A full cyber kill-chain simulation in an isolated VMware lab — Kali Linux as the attacker, Windows 11 as the victim — paired with real-time endpoint telemetry via Microsoft Sysmon and a SOC-style detection write-up mapped to MITRE ATT&CK.

What I built

  • Designed an isolated VMware lab with Kali Linux (attacker) and a Windows 11 endpoint (victim) on a host-only network to safely emulate adversary behavior.
  • Generated a TCP/IP reverse-shell payload with msfvenom, delivered it to the victim, and established a Meterpreter session via Metasploit's multi-handler.
  • Performed post-exploitation enumeration — system info, user context, network state — to demonstrate realistic attacker tradecraft after initial access.
  • Deployed Microsoft Sysmon with the SwiftOnSecurity configuration on the Windows endpoint to produce high-fidelity process, network, and registry telemetry.

How I detected it

  • Captured 985 endpoint events across the kill chain and analyzed them in Event Viewer to surface Indicators of Compromise.
  • Mapped IOCs to MITRE ATT&CK techniques and authored corresponding SIEM-style detection rules (process anomalies, suspicious child processes, non-standard ports).
  • Documented findings in a SOC-style incident response report covering timeline reconstruction, detection logic, and recommended hardening.
  • Validated detection coverage end-to-end — every step of the simulated intrusion produced at least one actionable telemetry artifact.
